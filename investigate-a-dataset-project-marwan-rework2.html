Introduction¶

Projecct Details: We are going to analyze a dataset and then communicate some insights and findings about it. We will use the Python libraries NumPy, pandas, Matplotlib and Seabork to make the analysis easier!

Dataset Used: In this project we are going to utilize a dataset called "TMDb movie data" from Kaggle.

Details about Dataset: This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue. Certain columns, like ‘cast’ and ‘genres’, contain multiple values separated by pipe (|) characters. The final two columns ending with “_adj” show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

Questions: We are going to use this data sets to deduce some conclusions and answer the following questions: Question 1: Which genres are more successful according to votes. Question 2: Which genres are more successful according to popularity. Question 3: Runtime & Voting relationship. Question 4: Genres co-relation with Budget & Revenue
